
# 📊 Customer Churn Prediction

## 📌 Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a customer is likely to leave a company based on customer behavior and account information. A synthetic dataset is generated within the program, and a Random Forest Classifier is used to classify customers as *Churn* or *Stay*.

---

## 🚀 Features

- Synthetic dataset generation
- Customer behavior analysis
- Random Forest Classification
- Churn prediction
- Interactive user input
- Save trained model
- Model evaluation

---

## ⚙️ Workflow

1. Generate customer dataset
2. Explore dataset
3. Split into training and testing data
4. Train Random Forest model
5. Evaluate the model
6. Save the trained model
7. Predict customer churn from user input
