
## 📌 Overview

This repository contains the Machine Learning projects completed as part of my *CodSoft Machine Learning Internship*. Each project demonstrates different Machine Learning concepts, including text classification, fraud detection, and customer behavior prediction.

The projects are implemented using *Python* and *Scikit-learn*, following a structured machine learning workflow from data generation and preprocessing to model training, evaluation, and prediction.

# 📁 Projects

## 🎬 Task 1 – Movie Genre Classification

Predicts the genre of a movie based on its plot using Natural Language Processing (NLP).

### Features

- Synthetic movie dataset generation
- TF-IDF Vectorization
- Logistic Regression
- Genre Prediction
- Model Evaluation

*Algorithms Used*

- TF-IDF Vectorizer
- Logistic Regression

---

## 💳 Task 2 – Credit Card Fraud Detection

Detects fraudulent credit card transactions using Machine Learning.

### Features

- Synthetic transaction dataset
- Fraud Classification
- Random Forest Classifier
- Model Evaluation
- User Prediction

*Algorithms Used*

- Random Forest Classifier

---

## 📊 Task 3 – Customer Churn Prediction

Predicts whether a customer is likely to leave a company based on customer information.

### Features

- Synthetic customer dataset
- Customer Churn Prediction
- Random Forest Classification
- Model Evaluation
- User Prediction

*Algorithms Used*

- Random Forest Classifier
