
# 💳 Credit Card Fraud Detection

## 📌 Overview

This project detects fraudulent credit card transactions using Machine Learning. A synthetic dataset is generated within the program, and a Random Forest Classifier is trained to classify transactions as Fraud or Legitimate.

---

## 🚀 Features

- Synthetic dataset generation
- Data preprocessing
- Random Forest Classification
- Fraud prediction
- Model evaluation
- Save trained model
- Interactive user prediction

---

## 🛠️ Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Joblib

## ⚙️ Workflow

1. Generate transaction dataset
2. Preprocess data
3. Split into training and testing sets
4. Train Random Forest model
5. Evaluate performance
6. Save trained model
7. Predict fraud using user input

---

## 📊 Evaluation Metrics

- Accuracy Score
- Classification Report
- Confusion Matrix
